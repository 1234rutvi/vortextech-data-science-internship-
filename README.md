# Vortex Tech Data Science Internship (2026)

## About
This repository contains the projects completed during my **Data Science & Analytics Internship** at **Vortex Tech**.

The internship focused on developing practical data science skills through hands-on projects involving **data preprocessing, exploratory data analysis, dashboard development, and machine learning** using Python and modern data analytics tools.

## Week 4 – Machine Learning using Random Forest Regressor

### Objectives
- Loaded and preprocessed the Superstore dataset
- Encoded categorical variables
- Split the dataset into training and testing sets
- Trained a Random Forest Regressor model
- Evaluated model performance using:
  - R² Score
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - Root Mean Squared Error (RMSE)
- Saved the trained model using Joblib

### Tools Used
- Python
- Pandas
- Scikit-learn
- Joblib
